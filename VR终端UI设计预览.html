头号空间 v1.3 — 纯沉浸式体验，零干扰 UI

设计原则：VR 终端不做任何可有可无的 UI。 用户戴上头盔的唯一目的就是沉浸式玩游戏。
所有操作（选游戏、支付、监控、结算）全部由 PC 终端 完成。 VR 终端只负责：加载过渡 → 纯游戏 → 结束提示（3 个必要条件）。

Logo 常亮浮空，30 秒后自动淡出至纯黑（省电）。用户无任何可交互元素。

最多显示 3 秒。游戏进程启动后自动切换至纯游戏画面。用户无需任何操作。
3 · GameRunning 游戏中 纯沉浸 · 0 UI
✦ 无倒计时 HUD
✦ 无系统菜单入口
✦ 无进度条叠加
✦ SDK 后台默默计时
完全沉浸。SDK 在后台做心跳 + 计时 + 摘盔检测，用户零感知。

5 · Error 系统异常 常驻直到处理
⚠️
